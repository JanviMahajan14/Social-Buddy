## Social-Buddy

This application is useful for students to create, post their notes. It also allows them to upvote and downvote the post. Also students can follow and unfolow people to get latest notifications.

## Development

### Backend

```sh
cd Social-Buddy
```

```sh
npm run dev
```

### Frontend

```sh
cd client
```

```sh
npm install
```

```sh
npm start
```

## API

```sh
/routes/auth
/post/:id
/users/:id
/user/profile/:id
```
